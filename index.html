Reflection on Grid

One thing i like about Grid..

what i really like about grid is that when we have to make and interactive website with a layout that contains multiple sections or say divs then placing them in the correct spot on the main Wrapper element becomes really easy and it is also much eaiser because the unit that we work with in the "fr" which makes it eaiser to map out all the elements so according to me making a website with alot of sections and divs and placing them in the correct spots is much easier with Grid so that's on thing i like about Css Grid.
